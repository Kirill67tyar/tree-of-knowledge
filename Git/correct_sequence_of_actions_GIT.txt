1. создаешь репозиторий на гитхабе
2. инициализируешь git на своём компе в нужной папке (обычно src)
3. подключаешь удалённый репозиторий, с тем что у тебя на компе
4. вытаскиваешь из удаленного ветку master --> git pull origin master
5. Всё. Теперь ты можешь делать изменения у себя на компе, когда будешь пушить изменения, они будут приходить в ветку master
