# хранилище данных, которое хранит данные в формате ключ значение
Предназнаено для быстрого ввода вывода
Позволяет хранить большое количество типов данных: строки, хэши, списки, кортежи и т.д.
Важно! - Redis хранит все в оперативной памяти, но мы можем настроить копирование данных на встроенные диски
с определенной переодичностью или при выполнении определенного действия
Полезно если нужно кэшировать данные
Своего рода альтернатива SQL если нужно работать с часто изменяющейся информацией

# можно:
# - задать данные SET key value

# - удалить через n секунд EXPIRE key n    (Где n - колиество секунд ввиде числа)
# - удалить в нужное время EXPIREAT key 00:00:00    (может формат времени пишется по другому)
