__init__.py - говорит о том, что это python package (модуль python)

wsgi.py - файл настройки для синхронного сервера (позволяет запустить django в СИНХРОННОМ режиме)

asgi.py - файл настройки для асинхронного сервера (позволяет запустить django в АСИНХРОННОМ режиме)

settings.py - основной файл, который сожержит в себе настройки проекта
