Немного размышлений - CRUD - базовый функционал
Для любого действия CRUD есть возможность работать как с одним объектом,
так и с множеством

Django admin предоставляет возможонсть реализовать свои действия для множества объектов
(для одного тоже)

Создать собственное действие - это значит создать функцию, которая принимает
указанные ниже аргументы:
 - объект ModelAdmin модели, которая отображается
 - объект запроса request типа HttpRequest
 - QuerySet объектов, которые выбрал пользователь
